About This Project

This Interactive AI Learning Hub was created as part of the IIT Jodhpur 2025 curriculum to provide students with engaging, hands-on learning experiences in Artificial Intelligence.

The platform combines theoretical knowledge with practical implementations, featuring interactive visualizations that make complex AI concepts accessible and understandable. Each topic is carefully designed with progressive learning in mind, from basic concepts to advanced implementations.

Built with modern web technologies including HTML5, CSS3, JavaScript, and mathematical rendering via MathJax, ensuring a smooth and educational user experience across all devices.
